# Plant Manager
**Application de gestion de plantes d’intérieur**

Plant Manager est une application web **full-stack** permettant de gérer ses plantes, suivre les tâches d’entretien, ajouter des notes et des photos, avec un système d’authentification sécurisé.

Realisé par : **Hajar Slimani**

---

## Stack Technique

### Backend
- Spring Boot 
- MySQL + JPA (Hibernate)
- Spring Security + JWT
- OAuth2 Google

### Frontend
- Angular 
- TypeScript
- Bootstrap + SCSS

---

## Fonctionnalités

### Authentification
- Inscription et connexion
- Connexion via Google OAuth2
- Routes sécurisées (JWT)

### Gestion des Plantes
- Ajouter, modifier et supprimer une plante
- Upload d’images avec compression automatique
- Affichage liste + détails

### Tâches d’Entretien
- Arrosage, fertilisation, etc.
- Fréquence configurable
- Calcul automatique de la prochaine date
- Marquer une tâche comme complétée

### Journal de Suivi
- Notes par plante
- Ajout de photos
- Historique chronologique
